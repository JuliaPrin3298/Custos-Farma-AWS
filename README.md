# Custos-Farma-AWS
Projeto prático focado na redução de custos de uma empresa hipotética por meio da otimização e escolha estratégica de serviços AWS.

# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

**Data:** 20/01/2026<br>
**Empresa:** Farma Rogers<br>
**Responsável:** Julia Rocha Nogueira

---

# Introdução

Este relatório apresenta o processo de implementação de serviços na empresa Farma Rogers, conduzido por Julia Rocha Nogueira.
O objetivo do projeto foi selecionar três serviços da AWS com foco na redução imediata de custos, mantendo a eficiência e a organização da infraestrutura.

# Descrição do Projeto

O projeto de implementação foi dividido em três etapas, cada uma com objetivos específicos.
A seguir, são descritas as etapas desenvolvidas ao longo do projeto, bem como suas finalidades e aplicações no contexto da empresa.

# Farma Rogers

A Farma Rogers é uma empresa farmacêutica que enfrenta dificuldades na gestão de custos, no armazenamento e organização de dados de clientes e fornecedores, além de não possuir uma infraestrutura de computação em nuvem implementada.

---

#Problemas identificados

### Gestão e Organização dos Lucros  
**Objetivo:** Otimização do controle financeiro e visualização dos lucros.

- [x] Armazenamento seguro de dados financeiros  
- [x] Alta disponibilidade dos dados  
- [x] Escalabilidade conforme crescimento da empresa  
- [x] Controle de acesso aos dados financeiros  
- [x] Possibilidade de integração com ferramentas de análise  
- [x] Redução de custos operacionais com infraestrutura  

**Melhor escolha:** Amazon QuickSight + Amazon S3

_______________________________________________________________________________________________

### Organização dos Dados dos Clientes
**Objetivo:** Centralizar e organizar os dados dos clientes de forma segura e eficiente.
- [x] Armazenamento estruturado de dados
- [x] Alta durabilidade e confiabilidade
- [x] Segurança e proteção de dados sensíveis
- [x] Facilidade de acesso e consulta
- [x] Backup automático
- [x] Conformidade com boas práticas de proteção de dados

 **Melhor Escolha:** Amazon RDS

_______________________________________________________________________________________________

### Gestão dos Fornecedores
**Objetivo:** Melhorar a organização e o controle das informações dos fornecedores.
- [x] Armazenamento centralizado das informações
- [x] Controle de permissões de acesso
- [x] Facilidade de atualização dos dados
- [x] Integração com outros sistemas da empresa
- [x] Escalabilidade da solução
- [x] Redução de falhas causadas por dados descentralizados

 **Melhor Escolha:** Amazon DynamoDB
_______________________________________________________________________________________________

Diante desse cenário, o projeto propõe a adoção de serviços AWS como solução para reduzir custos operacionais, melhorar a gestão de dados e modernizar a infraestrutura tecnológica da empresa.<br>

| Etapa | Serviço AWS | Objetivo |
|------|-------------|----------|
| 1 | QuickSight + S3 | Gestão financeira |
| 2 | RDS | Dados de clientes |
| 3 | DynamoDB | Fornecedores |


## Conclusão

A implementação dos serviços em nuvem da AWS na empresa Farma Rogers teve como finalidade a otimização da gestão de custos, a organização e segurança das informações e a melhoria da eficiência operacional. A adoção dessas soluções tecnológicas possibilita maior controle financeiro, além de proporcionar uma gestão mais estruturada dos dados de clientes e fornecedores. <br>
Com a utilização de uma infraestrutura em nuvem, a empresa passa a contar com recursos escaláveis, seguros e de alta disponibilidade, contribuindo para a modernização dos processos internos e para a tomada de decisões mais assertivas.<br>
Dessa forma, recomenda-se a continuidade da utilização dos serviços implementados, bem como a avaliação constante de novas tecnologias e soluções em nuvem que possam ampliar a eficiência, a confiabilidade e a competitividade da empresa no mercado.
